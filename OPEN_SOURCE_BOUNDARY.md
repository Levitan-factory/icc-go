# Open Source Boundary

This repository contains the local-first ICC-GO notebook application and the ICC DSL runtime it needs.

Excluded from this public export:

- Hosted OAuth configuration and callback handling.
- Hosted telemetry/admin dashboard implementation.
- AWS, Cloudflare, and production deployment scripts.
- Public marketing sites for ICC DSL and ICC-GO.

The public app stores notebooks, provider keys, attachments, and generated artifacts locally in the user's browser.
